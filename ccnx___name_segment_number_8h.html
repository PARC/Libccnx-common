Detailed Description

Utilities for creating and parsing CCNxNameSegments that contain an integer value.

CCNxNameSegments may optionally contain an integer value. This file contains some utilities for creating and parsing CCNxNameSegments with integer values. See CCNxNameSegment, CCNxNameLabelType, and CCNxName for more information.
