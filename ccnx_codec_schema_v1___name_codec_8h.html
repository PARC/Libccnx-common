CCNxName* ccnxCodecSchemaV1NameCodec_Decode ( CCNxCodecTlvDecoder *  decoder,
uint16_t  type 
)

Decode the buffer as a CCNxName beginning at the current position.

The buffer must be pointing to the beginnig of the "type". The decoder will verify that the type matches `type'. If it does not match, it will return NULL.

Parameters
[in]decoderThe decoder
[in]typeThe TLV type that the decoder should currently be pointing at
Returns
non-null The CCNxName decoded
null An error: either type did not match or some other error
