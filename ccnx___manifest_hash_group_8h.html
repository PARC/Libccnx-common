bool ccnxManifestHashGroup_Equals ( const CCNxManifestHashGroup *  x,
const CCNxManifestHashGroup *  y 
)

Determine if two CCNxManifestHashGroup instances are equal.

The following equivalence relations on non-null CCNxManifestHashGroup instances are maintained:

  • It is reflexive: for any non-null reference value x, CCNxManifestHashGroup_Equals(x, x) must return true.
  • It is symmetric: for any non-null reference values x and y, CCNxManifestHashGroup_Equals(x, y) must return true if and only if CCNxManifestHashGroup_Equals(y x) returns true.
  • It is transitive: for any non-null reference values x, y, and z, if CCNxManifestHashGroup_Equals(x, y) returns true and CCNxManifestHashGroup_Equals(y, z) returns true, then CCNxManifestHashGroup_Equals(x, z) must return true.
  • It is consistent: for any non-null reference values x and y, multiple invocations of CCNxManifestHashGroup_Equals(x, y) consistently return true or consistently return false.
  • For any non-null reference value x, CCNxManifestHashGroup_Equals(x, NULL) must return false.
Parameters
[in]xA pointer to a CCNxManifestHashGroup instance.
[in]yA pointer to a CCNxManifestHashGroup instance.
Returns
true if the referenced CCNxManifestHashGroup instances are equal.

Example:

1 {
2  CCNxManifestHashGroup *a = CCNxManifestHashGroup_Create(...);
3  CCNxManifestHashGroup *b = CCNxManifestHashGroup_Create(...);
4 
5  if (CCNxManifestHashGroup_Equals(a, b)) {
6  // true
7  } else {
8  // false
9  }

Release a previously acquired reference to the specified instance, decrementing the reference count for the instance.
